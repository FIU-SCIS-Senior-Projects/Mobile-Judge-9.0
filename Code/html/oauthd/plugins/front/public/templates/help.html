To use oauthd, you need to configure one or several apps, which represent your real world app. You will be able to add and configure various providers, and set the app mode so that you can call the authentication and requests endpoints to use the providers' APIs.

Using oauthd in a client app

You can either directly use the HTTP API, described here, or use one of the SDKs:

Front-end SDKs
  • JavaScript SDK
  • Phonegap SDK
  • iOS SDK
  • Android SDK
Back-end SDKs
  • NodeJs SDK
  • PHP SDK
  • Go SDK

When using these SDKs, you'll have to specify that you're using your own oauthd instance. For example in JavaScript: 

OAuth.setOAuthdURL('{{ config.host_url }}')

For more information about oauthd, please check out our WIKI and github repository
